# Continuity Rules

Validation rules applied by Confucius.

## Wardrobe (WARD)

| Rule | Description |
|------|-------------|
| WARD-01 | Wardrobe state changes require cause beat |
| WARD-02 | Conflicting wardrobe in adjacent timeline |
| WARD-03 | Signature items persist |

## Props (PROP)

| Rule | Description |
|------|-------------|
| PROP-01 | Prop cannot appear without introduction |
| PROP-02 | Ownership transfer must be shown |
| PROP-03 | Damage persists |

## Lighting (LGT)

| Rule | Description |
|------|-------------|
| LGT-01 | Time-of-day vs described lighting mismatch |
| LGT-02 | Weather continuity mismatch |
| LGT-03 | Match-cut consistency warnings |

## Timeline (TIME)

| Rule | Description |
|------|-------------|
| TIME-01 | Impossible travel given location distance |
| TIME-02 | Unresolved relative phrases |
| TIME-04 | Character in two places at once |

## Knowledge (KNOW)

| Rule | Description |
|------|-------------|
| KNOW-01 | Character cannot reference unlearned information |
| KNOW-02 | Secrets propagate only through shown channels |
| KNOW-03 | Motive/goal consistency |
| KNOW-04 | Relationship continuity |
